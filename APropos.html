Pourquoi un site Portfolio ?

Ce site web a pour but de montrer les différents projets réalisés et donc par conséquent ce dont je suis capable.
Cependant, pour l'heure le site n'est pas là pour montrer mes talents de développeur web car j'ai monter ce projet et appris le développement web seul.
Le site n'est donc qu'un projet personnel qui me sert d'interface entre vous et moi. J'ai tout de même fais de mon mieux pour rendre la navigation agréable alors j'espère qu'elle vous plaira.


Vous appréciez ce site web ?

Faites-le moi savoir ! mes contacts sont en haut à droite de l'écran. À l'inverse si vous n'aimez pas le site ou une partie du site pour x raison, dites-le moi aussi, je serai ravi d'entendre vos arguments et ainsi me permettre de faire évoluer le projet.
